商品 VIP 购买系统（C++ 课程大作业）

说明：这是一个简化的商品 VIP 购买系统，展示了面向对象（继承、封装、多态）、STL 算法、使用 `fork` 启动子进程运行各个接口，以及使用 POSIX 系统调用 `open/read/write/close` 做数据持久化。

构建：
  make

运行：
  ./bin/vip_system

数据文件：
  data/products.txt  - 商品列表（示例）
  data/users.txt     - 用户列表（示例）

简单说明：主进程显示菜单，每个选项会 `fork()` 启动子进程执行对应接口（查询、添加、购买、升级 VIP 等）。子进程通过系统调用读写数据文件，操作完成后退出，父进程等待并在需要时重新加载数据。
